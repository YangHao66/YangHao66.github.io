Hexo的安装

安装nodejs与git(已安装可以跳过)

在安装hexo前,我们要确保电脑中已经安装了node.js和Git.(点击即可下载)

在安装node.js时需要注意在Custom Setup这一步记得选 Add to PATH ,这样你就不用自己去配置电脑上环境变量了，否则的话会很麻烦.

安装完成后再按 win + r 快捷键调出运行，然后输入cmd确定 在命令提示窗口中输入 node -v 和git –version检测,如果出现版本信息说明安装成功.
